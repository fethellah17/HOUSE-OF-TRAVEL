## 🎬 ANIMATION TIMELINE

### Page Load (Hero Section)
```
0ms:    Hero container appears (hidden)
0ms:    Eyebrow fades in + moves up
150ms:  Headline fades in + moves up
300ms:  Subheadline fades in + moves up
450ms:  CTA buttons fade in + moves up
```

### Scroll Animations (Services Section)
```
When section enters viewport:
0ms:    Section header fades in + moves up
0ms:    Card 1 starts animation (hidden)
150ms:  Card 2 starts animation (hidden)
300ms:  Card 3 starts animation (hidden)

Each card animation:
- Opacity: 0 → 1
- Y position: 30px → 0px
- Duration: 600ms
```

### Scroll Animations (Offers Section)
```
When section enters viewport:
0ms:    Section header fades in + moves up
100ms:  Category filter fades in + moves up
0ms:    Trip card 1 starts animation
150ms:  Trip card 2 starts animation
300ms:  Trip card 3 starts animation
450ms:  Trip card 4 starts animation
600ms:  Trip card 5 starts animation
750ms:  Trip card 6 starts animation
```

### Hover Animations
```
Service Cards:
- Border color: slate-200 → Sage Green (300ms)
- Shadow: none → shadow-2xl (300ms)
- Icon scale: 1 → 1.1 (300ms)
- Bottom bar: scale-x-0 → scale-x-1 (300ms)

CTA Buttons:
- Scale: 1 → 1.02 (300ms)
- Shadow: default → shadow-2xl (300ms)
- Icon translate-x: 0 → 4px (300ms)
- Background: default → darker shade (300ms)
```

## 🎨 DESIGN PATTERNS

### Card Pattern
```
Structure:
1. Icon container (gradient background)
2. Title (Navy Blue, bold)
3. Description (Gray, regular)
4. Bottom accent bar (appears on hover)

Spacing:
- Icon to title: mb-6 (24px)
- Title to description: mb-3 (12px)
- Card padding: p-8 (32px)

Interaction:
- Hover: Border color change, shadow, icon scale, bottom bar
- Transition: 300ms ease
```

### CTA Pattern
```
Structure:
1. Text label
2. Icon (arrow or ticket)

Spacing:
- Text to icon: gap-3 (12px)
- Horizontal padding: px-8 to px-10 (32px - 40px)
- Vertical padding: py-4 to py-5 (16px - 20px)

Interaction:
- Hover: Scale up, shadow, icon translate, background darken
- Active: Scale down
- Transition: 300ms ease
```

### Section Header Pattern
```
Structure:
1. Eyebrow (Sage Green, uppercase)
2. Headline (Navy Blue, large)
3. Optional description (Gray)

Spacing:

- Headline to description: mb-4 to mb-6 (16px - 24px)
- Section header to content: mb-10 to mb-16 (40px - 64px)
```

## 📊 PERFORMANCE CONSIDERATIONS

### Animation Performance
```
- Use transform and opacity (GPU-accelerated)
- Avoid animating layout properties
- viewport={{ once: true }} prevents re-animation
- Stagger delays keep animations smooth
```

### Image Optimization
```
- No heavy background images
- SVG icons (scalable, small file size)
- Pattern overlays use CSS (no images)
```

### Code Splitting
```
- Components lazy-loaded where appropriate
- Framer Motion tree-shaken
- Lucide icons imported individually
```
