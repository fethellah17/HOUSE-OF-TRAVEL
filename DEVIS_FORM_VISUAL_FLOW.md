## Animation Timeline

```
User Action                Animation Effect              Duration
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Page Load              → Fade in + Slide up              0.3s
Select Service Path    → Scale + Shadow                  0.2s
Path Module Appears    → Fade in + Height expand         0.4s
Select Travel Type     → Scale + Background change       0.2s
Config Section Opens   → Fade in + Height expand         0.4s
Canada Magic Appears   → Scale + Spring bounce           0.5s
Document List Appears  → Staggered fade in (0.05s each)  Variable
Button Hover           → Scale 1.02 + Shadow             0.3s
Button Click           → Scale 0.98                      0.1s
Form Submit            → Loading spinner                 1.5s
Success Message        → Fade in + Scale                 0.3s
```
